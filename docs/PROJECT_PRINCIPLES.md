# FB Software AI - Project Principles

## Our Mission

Build the best open-source AI platform for WordPress and beyond.

## Core Principles

### 1. Documentation First
Every feature must be documented before it is implemented.

### 2. Simple Before Clever
Choose clear, maintainable solutions over unnecessary complexity.

### 3. Modular Design
Every module should be replaceable without affecting the rest of the system.

### 4. Open Source
The project should be transparent, welcoming, and community-driven.

### 5. Security
Protect user data and API keys by design.

### 6. Learning
Every feature should be understandable and teachable.
