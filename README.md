# 🤖 FB Software AI

> The open-source AI assistant for WordPress creators, agencies, educators, and businesses.

---

## 🚀 Vision

FB Software AI helps WordPress users by answering questions using their own knowledge base.

Instead of generic AI responses, it learns from:

- 📺 YouTube videos
- 📖 Blog posts
- 📄 PDFs
- 📚 Documentation
- 🎓 Courses
- ❓ FAQs
